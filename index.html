Informações sobre o site

Funcionalidades do site:

  • Extrair links de grupos do WhatsApp a partir de um texto copiado.
  • Remover links duplicados automaticamente.
  • Copiar os links extraídos para a área de transferência.
  • Alternar entre tema claro e escuro para uma melhor experiência visual.

Como usar:

  1. Copie o texto que contém os links de grupos do WhatsApp.
  2. Clique no botão "Colar" para colar o texto na área de entrada.
  3. Clique no botão "EXTRAIR LINKS DE GRUPOS" para extrair os links.
  4. Os links extraídos aparecerão na área de saída.
  5. Use o botão "Copiar" para copiar os links extraídos para a área de transferência.
  6. Use o botão "Limpar Tudo" para limpar todas as áreas de texto.
  7. Clique no botão "Alternar Tema" para mudar entre o tema claro e escuro.
